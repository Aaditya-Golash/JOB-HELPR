# Cover letter

Primary path: call `job-pipeline-mcp`'s `generate_cover_letter` tool — already
calibrated to the account's format (bolded thematic subheadings tied to the
JD, mirrors the employer's own language, short plain closing paragraph
instead of an extended summary).

Run `tailor-resume` first if a tailored resume for this posting doesn't exist
yet — the cover letter should reference the same framing, not a generic one.

## Voice rules (converged from all four source projects)

- First person, no em dashes.
- Achievement-focused: connect 2-3 measurable results to what the posting
  specifically asks for, not a generic list of strengths.
- Strict honesty — never fabricates or exaggerates anything not already in
  the resume/work history.
- Mirror the employer's own language from the posting where it's genuinely
  accurate, not as a keyword-stuffing exercise.
